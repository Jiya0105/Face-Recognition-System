## 📌 Overview  
A real-time **Face Recognition System** built using Python that detects and recognizes human faces using webcam input and stored images. This project demonstrates practical implementation of computer vision, face encoding, and image comparison techniques.

---

## ✨ Features  
- 🎥 Real-time face recognition via webcam  
- 🖼️ Face comparison using stored images  
- ⚡ Fast and efficient face encoding  
- 📂 Organized image dataset handling  
- 💻 Simple and easy-to-run scripts

## ⚙️ Setup & Installation  

### 1. Clone Repository  
git clone https://github.com/Jiya0105/Face-Recognition-System.git  
cd Face-Recognition-System  

### 2. Install Dependencies  
pip install opencv-python numpy face-recognition  

> Note: face-recognition may require cmake and dlib installed.

---

## ▶️ How to Run  

### 🔹 Real-time Face Recognition  
python main_video.py  

### 🔹 Compare Two Images  
python image_comparison.py  

---

## 📸 How It Works  
1. Load known images from images/ folder  
2. Encode faces using face recognition library  
3. Capture video from webcam  
4. Detect and match faces in real-time
